

## Проблема

Таймаут 5 минут (`MAX_PROCESSING_TIME = 300000ms`) недостаточен для генерации 6 карточек с единой стилизацией. Каждая карточка занимает ~60 сек + 6 сек пауза + ~10 сек на анализ стиля = ~7 минут на 6 карточек. Функция использует `waitUntil()` для фоновой работы, что позволяет превысить стандартный лимит edge function.

## Решение

Увеличить `MAX_PROCESSING_TIME` с 5 до 9 минут (540 секунд). Функция уже использует `EdgeRuntime.waitUntil()`, что позволяет фоновой задаче работать дольше стандартного лимита ответа. 9 минут покроет даже самый медленный сценарий (6 карточек + стиль + ретраи).

### Изменение

**Файл:** `supabase/functions/process-generation-tasks-banana/index.ts`, строка 65:

```
// Было:
const MAX_PROCESSING_TIME = 5 * 60 * 1000; // 5 minutes timeout

// Станет:
const MAX_PROCESSING_TIME = 9 * 60 * 1000; // 9 minutes timeout (waitUntil extends beyond default limit)
```

Затем деплой функции.
